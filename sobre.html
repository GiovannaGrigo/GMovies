Propósito

O site foi desenvolvido com o objetivo de apresentar ao usuário uma seleção dos filmes mais populares do momento, ajudando-o a descobrir novos títulos que possam despertar seu interesse. Além de fornecer uma lista atrativa, o site disponibiliza páginas dedicadas ao detalhamento de cada filme, onde o usuário pode explorar informações como os gêneros cinematográficos, a data de lançamento, o país de origem, a popularidade e uma sinopse envolvente.

Para enriquecer ainda mais a experiência, há também trailers que oferecem uma prévia do conteúdo do filme, ajudando o usuário a decidir se é algo que combina com seu gosto. Outro diferencial importante é a possibilidade de conhecer o elenco, com informações sobre os atores e atrizes que dão vida aos personagens, proporcionando uma visão completa do universo de cada obra.

Desenvolvimento

Esse projeto foi feito nas aulas de programação do professor Gallo no curso Barracred Conecta, onde conhecemos mais sobre integração de API utilizando o site "The Movie Data Base" e também tivemos uma maior familiarização com a linguagem de programação JavaScript. O site foi desenvolvido em HTML, CSS e JS, com o uso do Bootstrap e API.
