Propósito

O site foi criado com o intuito de mostrar ao usuário quais são os filmes mais procurados do momento, para que o usuário possa se interessar em assistir algum dos filmes apresentados. Com páginas de detalhamento de cada filme, o usuário poderá saber quais são os gêneros, a data de lançamento, o país de origem, a popularidade e a sinopse de cada filme, há também os trailers para que seja possível ver uma prévia do que será encontrado no filme e algo que é considerado muito importante, que é poder saber quais atores e atrizes fazem parte do elenco do filme.

Desenvolvimento

Esse projeto foi feito nas aulas de programação do professor Gallo, no curso Barracred Conecta, onde conhecemos mais sobre integração de API utilizando o site "The Movie Data Base" e também tivemos uma maior familiarização com a linguagem de programação JavaScript. O site foi desenvolvido em HTML, CSS e JavaScript, com utilização de Bootstrap e API.
